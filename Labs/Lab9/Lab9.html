Inline CSS

It allows the user to apply style rules to a specific HTML element. To use the inline styles, the style attribute has to be added to the relvant element. This style attribute can contain any CSS property. It can be useful when wanting to add style to you HTML elements such as color and fonts etc.

Internal CSS

An internal css can be used when there is one single HTML page that has a unique style. The internal style is defined inside the style element in the head section. They are defined withing the style command and are important when wanting to style a single HTML page.

External CSS

With an extrnal style sheet the look of an entire website just by changing one file. It is a seperate CSS file that can accsessed by creating a link withing the head section of a webpage. Each HTML page must include a reference to to the external style sheet inside the "link" element inside the head section.
